## 如何设置以及启动项目

### 安装依赖

```bash
npm install
```

### 启动本地开发环境（自带热启动）

```bash
npm run serve
```

### 构建生产环境 (自带压缩)

```bash
npm run build
```

### 代码格式检查以及自动修复

```bash
npm run lint
```

### 运行单元测试

```bash
npm run test:unit
```

### 运行端对端测试

```bash
npm run test:e2e
```

### 自动生成 svg 组件

```bash
npm run svg
```

### 自定义 Vue 配置

看这里 [Configuration Reference](https://cli.vuejs.org/config/).
